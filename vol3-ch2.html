2-1　平方根的意義

正負平方根 ‧ 根號的估算與比大小 ‧ √(a²)=|a|

選擇2-1-1📄 會考·111年🖨 第 1 頁
√2022 的值介於下列哪兩個數之間？
(A) 25，30
(B) 30，35
(C) 35，40
(D) 40，45
✎ 計算 / 作答區
✅ 答案：(D) 40，45
解析：∵ (√2022)²=2022
40²=1600，45²=2025
∴ 40<√2022<45，
故選 (D)。
列印第 2 頁 / 共 25 頁　·　2-1 平方根的意義
選擇2-1-2📄 改編·中小學題庫🖨 第 2 頁
下列有關平方根的敘述，何者正確？
(A) √16 = ±4
(B) -5 的平方根是 -5
(C) √((-7)²) = 7
(D) 9 的平方根是 3
✎ 計算 / 作答區
✅ 答案：√((-7)²) = 7
解析：逐一檢驗四個選項。
√16 表示 16 的正平方根，√16 = 4，並非 ±4。
-5 為負數，負數沒有實數平方根。
√((-7)²) = √49 = 7（即 |-7| = 7），正確。
9 的平方根有兩個，為 ±3，不能只寫 3。
故正確選項為 √((-7)²) = 7。

選擇2-1-3📄 改編·中小學題庫🖨 第 3 頁
設 √73 的整數部分為 a，小數部分為 b，則 2a + b 之值為何？
(A) √73
(B) 8 + √73
(C) 16 + √73
(D) 24 - √73
✎ 計算 / 作答區
✅ 答案：8 + √73
解析：先估算 √73 落在哪兩個連續整數之間。
因為 8² = 64、9² = 81，且 64 < 73 < 81，
所以 8 < √73 < 9，整數部分 a = 8。
小數部分 b = √73 - 8。
2a + b = 2×8 + (√73 - 8) = 16 + √73 - 8 = 8 + √73。

選擇2-1-4📄 改編·中小學題庫🖨 第 4 頁
將 3√2、2√5、√17、4 四數由小到大排列，正確的是？
(A) 4 < √17 < 3√2 < 2√5
(B) 4 < 3√2 < √17 < 2√5
(C) √17 < 4 < 3√2 < 2√5
(D) 4 < √17 < 2√5 < 3√2
✎ 計算 / 作答區
✅ 答案：4 < √17 < 3√2 < 2√5
解析：四數皆為正數，可將各數化為根號形式，再比較被開方數。
3√2 = √(9×2) = √18；2√5 = √(4×5) = √20；
√17 = √17；4 = √16。
比較被開方數：16 < 17 < 18 < 20，
故 4 < √17 < 3√2 < 2√5。
列印第 5 頁 / 共 25 頁　·　2-1 平方根的意義
選擇2-1-5📄 改編·中小學題庫🖨 第 5 頁
已知一數 x 滿足 x² = 11，下列關於 x 的敘述何者正確？
(A) x 只能是 √11
(B) x = ±√11
(C) x = 11/2
(D) 這樣的 x 不存在
✎ 計算 / 作答區
✅ 答案：x = ±√11
解析：x² = 11 表示要找「平方後等於 11」的數。
因為 11 是正數，這樣的數有兩個：一正一負。
正的一個記作 √11，負的一個記作 -√11。
驗算：(√11)² = 11、(-√11)² = 11，兩者皆成立。
所以 x = ±√11。
常見錯誤是只寫 √11，漏掉負根；而 11/2 是把「平方」誤當成「除以 2」。

選擇2-1-6📄 改編·中小學題庫🖨 第 6 頁
√54 介於下列哪兩個連續整數之間？
(A) 5 與 6 之間
(B) 6 與 7 之間
(C) 7 與 8 之間
(D) 8 與 9 之間
✎ 計算 / 作答區
✅ 答案：7 與 8 之間
解析：找出 54 落在哪兩個「完全平方數」之間。
7² = 49、8² = 64，而 49 ＜ 54 ＜ 64。
對三者同時開平方（根號愈開愈大不改變大小順序）：
√49 ＜ √54 ＜ √64，即 7 ＜ √54 ＜ 8。
所以 √54 介於 7 與 8 之間。

列印第 8 頁 / 共 25 頁　·　2-1 平方根的意義
選擇2-1-8📄 改編·中小學題庫🖨 第 8 頁
將 2√3、3√2、√20、5 四數由小到大排列，正確的是？
(A) 2√3 ＜ 3√2 ＜ √20 ＜ 5
(B) √20 ＜ 2√3 ＜ 3√2 ＜ 5
(C) 2√3 ＜ √20 ＜ 3√2 ＜ 5
(D) 3√2 ＜ 2√3 ＜ 5 ＜ √20
✎ 計算 / 作答區
✅ 答案：2√3 ＜ 3√2 ＜ √20 ＜ 5
解析：把每個數都改寫成「單一根號」的形式，再比較根號內的數。
2√3 = √(2²×3) = √12
3√2 = √(3²×2) = √18
√20 = √20
5 = √(5²) = √25
根號內由小到大：12 ＜ 18 ＜ 20 ＜ 25。
因此 2√3 ＜ 3√2 ＜ √20 ＜ 5。
列印第 9 頁 / 共 25 頁　·　2-1 平方根的意義
計算2-1-9📄 改編·中小學題庫🖨 第 9 頁
化簡 √((-11)²) + √((4-9)²) - √(3²) 之值。
✎ 計算 / 作答區
✅ 答案：13
解析：利用性質 √(a²) = |a|（開完平方一定是非負數，等於絕對值）。
√((-11)²) = |-11| = 11。
先算括號：4 - 9 = -5，故 √((-5)²) = |-5| = 5。
√(3²) = |3| = 3。
合併：11 + 5 - 3 = 13。
提醒：√((-11)²) 不可寫成 -11，因為根號的結果不會是負數。

化簡 √32 + √18 - √8 之值為何？
(A) √42
(B) 3√2
(C) 5√2
(D) 9√2
✎ 計算 / 作答區
✅ 答案：5√2
解析：先把每一項化成「最簡根式」，抽出完全平方因數。

列印第 21 頁 / 共 25 頁　·　2-3 畢氏定理

2-3　畢氏定理

直角三角形三邊關係 ‧ 平面兩點距離 ‧ 畢氏定理的應用

選擇2-3-1📄 改編·中小學題庫🖨 第 21 頁
如圖，直角三角形的兩股長分別為 9 與 12，則斜邊長為何？
(A) 13
(B) 15
(C) 18
(D) 21
✎ 計算 / 作答區
✅ 答案：15
解析：設斜邊長為 c，由畢氏定理：兩股平方和等於斜邊平方。
9² + 12² = c²，
81 + 144 = 225 = c²，
c = √225 = 15。
驗算：9² + 12² = 81 + 144 = 225 = 15²，成立。
列印第 22 頁 / 共 25 頁　·　2-3 畢氏定理
選擇2-3-2📄 改編·中小學題庫🖨 第 22 頁
下列各組長度（單位相同）中，哪一組可以構成直角三角形？
(A) 5、6、7
(B) 4、5、6
(C) 8、15、17
(D) 6、7、9
✎ 計算 / 作答區
✅ 答案：8、15、17
解析：三邊能構成直角三角形的條件：兩短邊平方和 = 最長邊平方。
檢驗 8、15、17：8² + 15² = 64 + 225 = 289，而 17² = 289，兩者相等，成立。
其餘各組：5² + 6² = 61 ≠ 49；4² + 5² = 41 ≠ 36；6² + 7² = 85 ≠ 81，皆不成立。
故選 8、15、17。
列印第 23 頁 / 共 25 頁　·　2-3 畢氏定理
選擇2-3-3📄 改編·中小學題庫🖨 第 23 頁
在座標平面上，A(-2, 1) 與 B(3, 13) 兩點間的距離為何？
(A) 11
(B) 12
(C) 13
(D) 17
✎ 計算 / 作答區
✅ 答案：13
解析：利用兩點距離公式（即畢氏定理）。
水平差 = 3 - (-2) = 5，鉛直差 = 13 - 1 = 12。
距離 = √(5² + 12²) = √(25 + 144) = √169 = 13。
驗算：5² + 12² = 25 + 144 = 169 = 13²，成立。
列印第 24 頁 / 共 25 頁　·　2-3 畢氏定理
選擇2-3-4📄 改編·中小學題庫🖨 第 24 頁
一支垂直於地面的旗桿高 8 公尺，桿頂拉一條繩子固定在地面上，固定點離桿底 6 公尺。若繩子拉直，則繩長為多少公尺？
(A) 9
(B) 10
(C) 12
(D) 14
✎ 計算 / 作答區
✅ 答案：10
解析：旗桿（8 公尺）與地面（6 公尺）互相垂直，繩子為斜邊。
繩長² = 8² + 6² = 64 + 36 = 100，
繩長 = √100 = 10（公尺）。
驗算：6² + 8² = 36 + 64 = 100 = 10²，即為 6-8-10 直角三角形，成立。
列印第 25 頁 / 共 25 頁　·　2-3 畢氏定理
選擇2-3-5📄 改編·中小學題庫🖨 第 25 頁
如圖，一個正方形的對角線長為 8，則此正方形的面積為多少？
(A) 16
(B) 32
(C) 32√2
(D) 64
✎ 計算 / 作答區
✅ 答案：32
解析：設正方形邊長為 a。對角線將正方形分成兩個等腰直角三角形。
由畢氏定理：對角線² = a² + a² = 2a²。
即 8² = 2a²，64 = 2a²，a² = 32。
面積 = a² = 32。
（也可用：面積 = 對角線² ÷ 2 = 64 ÷ 2 = 32。）
此即邊長為對角線的 1/√2 倍、對角線為邊長的 √2 倍之關係。
